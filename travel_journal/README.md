# Travel Journal 🌍

A simple React app that displays a journal of travel destinations. Built while learning React fundamentals.

## About

A static travel journal app showcasing hardcoded travel destinations with details like location, dates, and description. Made as a practice project while following a React course.

## Tech Stack

- React (Vite)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ImSatyamK/react-learning.git
cd react-learning/travel_journal
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```

## Notes

- This is a beginner project made for learning purposes
- Data is hardcoded, no API or database involved
